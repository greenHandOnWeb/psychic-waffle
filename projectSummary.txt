作品总结

作品名称
灵动画册

作品介绍
灵动画册是一款面向日常分享与素材管理的 Web 应用，支持图片上传与画廊浏览、基于 Fabric 的画布拼图编辑、云端模板套用、拼图成品导出与带时间轴的音频海报 HTML 导出，以及将多张作品编排为视频的流程（结合浏览器能力）。默认可配合 Supabase 做登录、存储与作品数据持久化，亦支持 Mock 模式便于本地开发。

系统核心能力包括：
- 画廊：单图 / 拼图分类、标签筛选与编辑、点赞、批量删除（与导出视频勾选共用）、本地上传与压缩。
- 编辑器：多图层图片与文字、贴纸、画布背景色 / 背景图、云端模板目录（JSON）与首次套用时的整幅栅格快照逻辑；布局以百分比写入 JSON，画布尺寸随容器动态适配。
- 导出：拼图 JPEG、海报图、含音频时间轴的 HTML；模板相关合成图可上传至存储桶。
- 视频：基于选中画廊图顺序的生成向导（与画布导出能力衔接）。
- AI 相关：Mock 或接入的标签、浏览器端抠图与图片增强等扩展点。

面向用户
喜欢拼图发社交动态、需要轻量相册与标签管理、希望用模板快速出图或导出带背景音乐海报的普通用户与内容创作者。

使用场景
旅行 / 活动照片整理与拼图发布；统一尺寸的活动海报；多图轮播式长图故事；为视频准备统一画幅的素材帧等。

作品价值
降低「多图排版 + 导出 + 分发」的工具切换成本；模板与布局 JSON 便于运营侧扩展样式而不改客户端逻辑；纯前端为主、可选后端托管，部署形态灵活。

后续产品方向（摘录）
- 云端模板：更长、更完整的竖版 / 分镜式模板（已在内置目录中增加「竖版分镜长图」示例，可继续扩展 JSON）。
- 海报：可演进为可配置表单 + 可分享链接形态。
- 回收站：批量删除软删除与彻底删除（依赖库表与存储策略）。
- 分发：PWA / 壳应用或国内镜像部署以改善访问；UI 组件库若统一为 ant-design-vue 需专项迁移。
