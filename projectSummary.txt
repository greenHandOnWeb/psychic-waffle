作品总结

作品名称

- 灵动画册
- 仓库目录名 psychic-waffle；

作品介绍

- 解决多图成页、保存与分享的需求，无需重型设计软件
- 上传图片后 Mock/AI 辅助打标签，画廊浏览与管理作品
- 浏览器内 Fabric 画布：云端模板或自由排版，生成拼图式海报
- 可导出静态海报、带配乐时间轴的 HTML、多图幻灯 WebM


系统核心能力包括：

- 画廊：单图/拼图筛选，标签多选，点赞，元数据编辑，批量删除
- 画廊：勾选作品进入视频向导（时间轴、配乐、导出 WebM）
- 编辑器：模板槽位、文字与表情包、底色与背景图、音频时间轴
- 编辑器：导出海报与含乐 HTML；保存并生成拼图成品记录
- 数据：迁移 SQL + RLS；经 Supabase SDK 访问，无自建业务后端

面向用户

- 需要快速做一页拼图海报的个人用户
- 需要轻量展示与互动（公开作品、点赞）的小团队
- 需在本地或 BaaS 上完成原型、答辩、作品集的学习者

使用场景

- 旅行、活动、产品多图整理后做分享页
- 社群或店铺统一风格出活动图
- 编辑器内试听配乐，导出可即开即播的 HTML
- 从画廊勾选素材生成幻灯短视频
- 无 Supabase 时用 Mock 完整演练上传与编辑

作品价值

- 验证「前端重交互 + BaaS」在中小型创意工具中的可行性
- 画布与存储分层清晰，便于后续接真实 AI 或商业化模板
- 可扩展独立后端而不推翻现有数据模型
- 保留离线/本地演示能力，适合作业答辩与二次开发基座
